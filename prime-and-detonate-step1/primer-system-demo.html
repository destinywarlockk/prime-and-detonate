Technical Implementation Details

  • Grid Layout: CSS Grid with 2 columns (50% each) and 2 balanced rows (1fr each)
  • Right Alignment: All content is right-aligned within columns with 6px right padding
  • Stacked Labels: Labels are positioned above values for better readability
  • Slot Dots: 11px diameter circles with element-specific colors and subtle glow effects
  • Tabular Numerals: Uses font-variant-numeric: tabular-nums to prevent digit jitter
  • Fixed Sizing: Content area maintains consistent dimensions to prevent layout shift
  • Element Colors: Pulled from existing game color tokens with subtle glow effects
  • Grunts: SH column is hidden using display: none for grunt enemies

Primer Logic

  • First Prime: Fills Slot 1 (left column, under SH)
  • Second Prime: Fills Slot 2 (right column, under HP)
  • Mixed Primes: Each slot displays the color of the element that filled it
  • Clear on Detonate: Both dots clear when primes are detonated or expire
  • Visual Feedback: Card border and glow match the first prime element
